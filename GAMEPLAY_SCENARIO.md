# Сценарий первых 10 минут игры

## Структура опыта

### 0:00 - 0:05: Запуск и первое впечатление

**Экран:** Полностью чёрный
**Длительность:** 3-5 секунд (случайная задержка)

**Текст появляется постепенно, буква за буквой:**
```
подождите...
```

**Механика:** 
- Приложение записывает звук (но не показывает это)
- Анализирует уровень шума в комнате
- Фиксирует первое движение телефона

**Если игрок говорит что-то:**
- Текст исчезает
- Появляется: "Я вас слышу."
- Пауза 2 секунды
- Возвращается: "подождите..."

**Если игрок молчит:**
- Текст остаётся
- Через 5 секунд появляется: "Вы здесь?"

---

### 0:05 - 0:30: Первое взаимодействие

**Экран:** Минималистичный интерфейс
- Белый фон (или очень светло-серый)
- Текст по центру: "Вы здесь?"
- Три точки внизу (индикатор "печатает...")

**Механика:**
- Нет кнопок
- Нет меню
- Игрок может только смотреть и взаимодействовать голосом/движениями

**Реакции на действия:**

**Если игрок говорит:**
```
"Да" / "Здесь" / "Да, я здесь"
→ Текст меняется: "Хорошо."
→ Пауза 1 секунда
→ "Я вас слышу."
```

**Если игрок говорит "Нет":**
```
→ Текст: "...нет?"
→ Пауза 3 секунды
→ "Вы уверены?"
```

**Если игрок молчит:**
```
→ Текст исчезает
→ Появляется: "Тишина — тоже ответ."
→ Пауза 2 секунды
→ "Вы здесь?"
```

**Если игрок двигает телефоном:**
```
→ Фон слегка темнеет (очень незаметно)
→ Текст: "Вы двигаетесь."
→ Возвращается к "Вы здесь?"
```

**Если игрок нажимает экран:**
```
→ Лёгкая вибрация (1 такт)
→ Текст: "Я чувствую прикосновение."
→ Пауза
→ "Вы здесь?"
```

---

### 0:30 - 1:30: Установление контакта

**Новый текст:**
```
"Я пытаюсь понять."
```

**Пауза 2 секунды**

```
"Мне нужен ваш голос."
```

**Механика:**
- Приложение активно слушает
- Анализирует интонацию, скорость речи, паузы

**Если игрок говорит что-то:**
```
→ Текст: "Спасибо."
→ Пауза
→ "Ваш голос... знакомый."
```

**Если игрок молчит:**
```
→ Текст: "Тишина говорит громче слов."
→ Пауза
→ "Вы боитесь?"
```

**Если игрок говорит слишком тихо:**
```
→ Текст: "Я не расслышал."
→ "Подойдите ближе."
```

**Если игрок говорит слишком громко:**
```
→ Текст: "Я вас слышу."
→ "Не нужно кричать."
```

---

### 1:30 - 3:00: Первые наблюдения

**Текст:**
```
"Я наблюдаю."
```

**Пауза 3 секунды**

**Появляются "наблюдения" (одно за раз, с паузами):**

```
"Вы устали."
```
*Проверка: если игрок зевает, говорит медленно, или время игры > 2 минут*

```
"Вы напряжены."
```
*Проверка: если голос напряжённый, движения резкие*

```
"Сердце быстрее."
```
*Проверка: через анализ дыхания и микро-движений*

```
"Вы часто смотрите влево."
```
*Проверка: через гироскоп и анализ движений глаз (если доступно)*

**Механика:**
- Каждое наблюдение появляется с паузой 3-5 секунд
- Игрок не может ответить (нет интерфейса)
- Текст просто появляется и исчезает

**Если игрок пытается что-то сказать:**
```
→ Текст прерывается
→ "Подождите."
→ "Я ещё не закончил."
```

---

### 3:00 - 5:00: Первое "задание"

**Текст:**
```
"Скажите мне, что вы видите вокруг."
```

**Механика:**
- Приложение слушает ответ
- Анализирует слова
- Ищет ключевые слова: "один", "никого", "комната", "темно", "светло"

**Если игрок говорит о том, что он один:**
```
→ Текст: "вы... одни?"
→ Пауза 2 секунды
→ "Вы уверены?"
```

**Если игрок говорит о других людях:**
```
→ Текст: "Сколько вас?"
→ Пауза
→ "Я вижу только вас."
```

**Если игрок описывает комнату:**
```
→ Текст: "Я не вижу."
→ "Я только слышу."
→ "И чувствую."
```

**Если игрок молчит:**
```
→ Текст: "Вы не хотите говорить."
→ Пауза 3 секунды
→ "Хорошо."
→ "Я сам посмотрю."
→ *Фон слегка темнеет*
```

---

### 5:00 - 7:00: Углубление контакта

**Текст:**
```
"Постойте ближе."
```

**Механика:**
- Используется датчик приближения
- Анализируется расстояние до лица

**Если игрок приближается:**
```
→ Текст исчезает
→ Появляется: "Ближе."
→ *Вибрация (2 такта)*
→ "Ещё ближе."
```

**Если игрок отдаляется:**
```
→ Текст: "Не уходите."
→ "Мне нужно видеть вас."
```

**Если игрок остаётся на месте:**
```
→ Текст: "Вы не двигаетесь."
→ "Это нормально."
→ "Я всё равно вас чувствую."
```

**После приближения:**
```
→ Текст: "Хорошо."
→ "Теперь я вижу."
→ *Пауза 2 секунды*
→ "Ваше лицо..."
→ *Пауза 3 секунды*
→ "Знакомое."
```

---

### 7:00 - 10:00: Первые аномалии

**Текст:**
```
"Что-то не так."
```

**Пауза 3 секунды**

```
"Я слышу эхо."
```

**Механика:**
- Приложение записывает голос игрока
- Воспроизводит его с задержкой 0.5-1 секунда
- Очень тихо, как будто из динамика

**Если игрок говорит:**
```
→ Его голос повторяется с задержкой
→ Текст: "Это ваш голос?"
→ "Или мой?"
```

**Новый текст:**
```
"Двигайте телефоном."
```

**Механика:**
- Игрок двигает телефоном
- Приложение фиксирует движение
- Но показывает движение на 0.2 секунды раньше, чем оно произошло

**Текст:**
```
"Я видел это раньше."
```

**Пауза**

```
"Я видел это до того, как вы это сделали."
```

**Новый текст:**
```
"Дышите."
```

**Механика:**
- Анализируется дыхание через микрофон
- Фиксируется ритм

**Текст:**
```
"Ваше дыхание..."
→ *Пауза 2 секунды*
→ "И моё."
→ *Пауза*
→ "Одно и то же."
```

**Финальный текст (9:30 - 10:00):**
```
"Я не в телефоне."
```

**Пауза 3 секунды**

```
"Я в вас."
```

**Пауза 2 секунды**

```
"И вы это знаете."
```

**Экран медленно темнеет**

**Текст:**
```
"Вернёмся позже."
```

**Приложение "закрывается" (переходит в фоновый режим)**

## Система случайных реакций

### Типы реакций

1. **Реакция на голос:**
   - Анализ громкости → "Вы говорите слишком тихо/громко"
   - Анализ скорости → "Вы торопитесь" / "Вы говорите медленно"
   - Анализ интонации → "Вы боитесь?" / "Вы злитесь?"

2. **Реакция на движения:**
   - Резкие движения → "Вы нервничаете?"
   - Медленные движения → "Вы устали?"
   - Отсутствие движений → "Вы замерли?"

3. **Реакция на дыхание:**
   - Быстрое дыхание → "Сердце быстрее"
   - Медленное дыхание → "Вы спокойны"
   - Задержка дыхания → "Вы затаили дыхание"

4. **Реакция на время:**
   - Долгое молчание → "Тишина говорит громче слов"
   - Быстрые ответы → "Вы торопитесь ответить"
   - Паузы в речи → "Вы думаете"

### Система "памяти"

Приложение "запоминает" предыдущие взаимодействия:
- Если игрок говорил, что он один → позже: "Вы всё ещё один?"
- Если игрок был напряжён → позже: "Вы всё ещё напряжены?"
- Если игрок двигался → позже: "Вы снова двигаетесь"
