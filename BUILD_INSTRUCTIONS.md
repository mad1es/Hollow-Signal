# Инструкции по сборке Hollow Signal

## Требования

- macOS с установленным Xcode 15.0 или новее
- iOS 17.0+ для запуска на устройстве
- Аккаунт Apple Developer (для запуска на реальном устройстве)

## Шаги сборки

### 1. Открытие проекта

```bash
cd "/Users/madi/Desktop/game design/sig nal"
open HollowSignal.xcodeproj
```

Или просто дважды кликните на `HollowSignal.xcodeproj` в Finder.

### 2. Настройка проекта

1. В Xcode выберите проект в навигаторе (левый верхний угол)
2. Выберите target "HollowSignal"
3. Перейдите на вкладку "Signing & Capabilities"
4. Выберите вашу команду разработчика (Team)
5. Убедитесь, что Bundle Identifier уникален (например, `com.yourname.hollowsignal`)

### 3. Настройка разрешений

Приложение требует следующие разрешения (уже настроены в Info.plist):
- **Микрофон** — для анализа звука и дыхания
- **Распознавание речи** — для понимания слов игрока
- **Камера** (опционально) — для будущего фейс-трекинга

При первом запуске iOS запросит эти разрешения автоматически.

### 4. Выбор устройства

В верхней панели Xcode выберите:
- **Симулятор iOS** — для тестирования (рекомендуется iPhone 14 Pro или новее)
- **Ваше устройство** — для реального тестирования (требует подключения по USB)

⚠️ **Важно:** Некоторые функции (гироскоп, датчик приближения) работают только на реальном устройстве!

### 5. Запуск

Нажмите **⌘R** или кнопку "Run" в Xcode.

### 6. Тестирование на реальном устройстве

1. Подключите iPhone/iPad через USB
2. Разблокируйте устройство
3. Доверьтесь компьютеру при появлении запроса
4. Выберите устройство в Xcode
5. Нажмите Run

При первом запуске на устройстве:
- iOS может показать предупреждение о недоверенном разработчике
- Перейдите в Настройки → Основные → VPN и управление устройством
- Выберите ваш профиль и нажмите "Доверять"

## Известные проблемы

### Симулятор
- Датчик приближения не работает
- Гироскоп работает ограниченно
- Распознавание речи может работать медленнее

### Реальное устройство
- Требуется активное подключение к интернету для первого запуска распознавания речи
- Может потребоваться ручная настройка разрешений в Настройках iOS

## Отладка

### Проверка разрешений

Если приложение не работает правильно:

1. Откройте Настройки iOS
2. Найдите "Hollow Signal" в списке приложений
3. Убедитесь, что включены:
   - Микрофон
   - Распознавание речи

### Логи

В Xcode откройте консоль (⌘⇧Y) для просмотра логов:
- Ошибки распознавания речи
- Проблемы с датчиками
- Общие ошибки приложения

## Производительность

Приложение оптимизировано для:
- iPhone 12 и новее
- iPad (8-го поколения) и новее

На более старых устройствах могут быть задержки в распознавании речи.
