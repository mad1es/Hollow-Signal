# Настройка API для LLM

## OpenAI API

Приложение использует OpenAI API для генерации ответов. Для работы необходимо настроить API ключ.

### Способ 1: Переменная окружения (рекомендуется)

1. В Xcode выберите схему (Scheme) → Edit Scheme
2. Перейдите на вкладку "Run"
3. В разделе "Environment Variables" добавьте:
   - Name: `OPENAI_API_KEY`
   - Value: ваш API ключ от OpenAI

### Способ 2: Прямая передача в код (не рекомендуется для продакшена)

Откройте `LLMService.swift` и измените:

```swift
init(apiKey: String? = nil) {
    self.apiKey = apiKey ?? "ваш-ключ-здесь"
}
```

### Получение API ключа

1. Зарегистрируйтесь на https://platform.openai.com
2. Перейдите в раздел API Keys
3. Создайте новый ключ
4. Скопируйте ключ (он показывается только один раз!)

### Альтернативные провайдеры

Если хотите использовать другой провайдер (Anthropic, Google, и т.д.), измените:
- `baseURL` в `LLMService.swift`
- Формат запроса в методе `sendMessage`
- Модель в параметре `model`

### Безопасность

⚠️ **НИКОГДА** не коммитьте API ключи в Git!
- Добавьте `.env` или файлы с ключами в `.gitignore`
- Используйте переменные окружения или secure storage
- Для продакшена используйте серверный прокси

### Тестирование без API

Если API ключ не настроен, приложение будет использовать fallback реакции (прямые текстовые ответы без LLM).
